## Serial values

**This values are the ones that are used for the Bluetooth** 

|  Number |       Purpose     |
|:-------:|:-----------------:|
|    0    |     Null Value    |
|    1    |     Laser Point   |   
|    2    |     Special Gun   |    
|    3    |     IR Point      |

###### Note that the Laser Point and the IR Point are points that we received because we were hit
