# Sketch Connections

Follow the next table to connect the Arduino Nano and the Oled Display.

| Display	      | Arduino Pin   |
|:-------------:|:-------------:|
|     D0	      |      12       |
|     D1	      |      11       |   
|     CS	      |      8        |
|     DC	      |      9        |
|     RES	      |      10       |
